# Run on Localhost

## Prerequisites
- Node.js installed
- MongoDB installed and running

## Quick Start

### 1. Start MongoDB
```bash
mongod
```

### 2. Start Backend (Terminal 1)
```bash
cd backend
npm install
npm start
```
Backend runs on: http://localhost:5000

### 3. Start Frontend (Terminal 2)
```bash
cd frontend
npm install
npm run dev
```
Frontend runs on: http://localhost:3000

## Done!
Open http://localhost:3000 in your browser.
