## ⚡ INSTANT DEPLOYMENT (5 MINUTES)

### Option 1: Deploy to Vercel + Render (RECOMMENDED - 100% FREE FOREVER)

#### Step 1: Deploy Frontend to Vercel (2 minutes)

**Click this button:**

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/dsa-platform/tree/master/frontend)

OR manually:

1. Go to https://vercel.com
2. Click "Add New" → "Project"
3. Import from GitHub: `dsa-platform`
4. Root Directory: `frontend`
5. Click "Deploy"

**Your frontend is now LIVE at:** `https://dsa-platform-xxx.vercel.app`

#### Step 2: Deploy Backend to Render (3 minutes)

**Click this button:**

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/YOUR_USERNAME/dsa-platform)

OR manually:

1. Go to https://render.com
2. Click "New" → "Web Service"
3. Connect GitHub: `dsa-platform`
4. Root Directory: `backend`
5. Build Command: `npm install`
6. Start Command: `node server.js`
7. Add Environment Variable:
   - `MONGODB_URI`: Get from MongoDB Atlas (see below)
8. Click "Create Web Service"

**Your backend is now LIVE at:** `https://dsa-platform-api.onrender.com`

#### Step 3: Setup MongoDB Atlas (FREE - 2 minutes)

1. Go to https://mongodb.com/cloud/atlas/register
2. Create FREE account
3. Click "Build a Database" → Choose FREE tier
4. Click "Create"
5. Create Database User:
   - Username: `dsauser`
   - Password: (generate secure password)
6. Network Access → "Add IP Address" → "Allow Access from Anywhere"
7. Click "Connect" → "Connect your application"
8. Copy connection string:
   ```
   mongodb+srv://dsauser:PASSWORD@cluster0.xxxxx.mongodb.net/dsa-platform
   ```
9. Paste this in Render Environment Variables as `MONGODB_URI`
